# Uz Currency Bot

Ushbu Telegram bot foydalanuvchilarga valyuta kurslarini real vaqtda taqdim etadi. Bot Python tilida yozilgan bo'lib, sodda va tez ishlashga mo'ljallangan.

## Imkoniyatlar
- USD, EUR, RUB va boshqa valyutalar kursi
- So'mga nisbatan kurs
- Tezkor javob

## Ishlatish
1. Botni oching @uz_currency_bot
2. /start buyrug'ini yuboring
3. Valyuta nomini yozing masalan USD, EUR, RUB
4. Bot kursni ko'rsatadi

## Texnologiyalar
- Python 3
- Aiogram 3
- ExchangeRate API
- Railway hosting
